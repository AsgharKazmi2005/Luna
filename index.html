How It Works

It doesn't matter if you are an artist trying to make an accurate painting, an aerospace engineer looking for accurate data, or a mathematician in an intensive math modeling competition, the Luna data calculator can cover your statistical needs. The Lunar Data Calculator works by using a combination of multiple APIs to collect Lunar data for a certain time and locations. Please enter a date and region to extract data. If an entered location does not exist in the respective API endpoints, the city with the closest name will be shown. If an entered location exists but has no data, the nearest city will be shown. Site not supported on all devices and browsers, Google Chrome is recommended. Calculations take 2-10 Seconds.

Data Parameters

The Data Provided includes the Moon Rise, Moon Set, Moon Altitude (Height from given region), Moon Distance (Distance from given region), Moon Azimuth (Cardinal direction of The Moon) and the Moon Parallactic Angle (angle between the horizon and the celestial meridian of a given observation point). Soon the calculator will support Moon Phase description and imagery among other data.
